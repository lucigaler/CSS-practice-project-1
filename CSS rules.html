CSS Rules

This page is designed to explain CSS rules and their functions, and to give some examples of how they are used with HTML pages.

First, we will look at CSS Selectors. Each selector will target different HTML elements, so it is important to know when it is appropriate to use each one.

CSS Selectors and their purpose
CSS Selector Name Description
p Element Selector This will style all p elements in the web page
.main Class Selector This will style all elements with the "main" class
#title ID Selector This will style the element with the ID "title"
p .main Element with Class This selector will target all p elements with the class "main"
.main p Element under Class This will target any p elements that are nested within an element that has the "main" class

Specificity is an important feature to know when using CSS selectors. It refers to which selectors have priority over others, so that if more than one style could apply to a particular element, the browser will know which one to use.

The general rule is that ID selectors are more important than class selectors, which in turn are more important than element selectors. Styling done directly in the HTML code, such as using the <style> tag or inline styles, will override most CSS rules. Using !important when defining a style will override any other styling, however, so this takes the #1 spot!

Now that we understand selectors and specificity, let's have a look at the properties we can assign styles with!

CSS Properties and Values
Property Values Description
color Maroon, Azure Changes the colour of the element targetted — CSS has 140 supported colour names
font-family Georgia, Arial Changes the typeface
font-size 12px Changes the font size — this example uses pixels, but relative units are also used
font-weight bold Changes the thickness of the letters — it is possible to use numerical values as well as bold or normal
text-align center Aligns the text either left, right, centered or justified
opacity 0 to 1 Changes the opacity of the text from 1 - fully opaque - to 0 - fully transparent

These are just a few examples of CSS properties that can be used to style a webpage. Take a look at this website for more!
